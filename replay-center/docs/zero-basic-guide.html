回放中心零基礎教學

車機上傳影片、產生一次性二維碼、手機掃碼觀看與下載。

一、同 Wi-Fi 測試一次性 QR

  1. 在電腦開啟 PowerShell。
  2. 輸入：
cd C:\Users\Administrator\ReplayCenter
npm run server:once

畫面會顯示上傳 API 和一次性 API。

二、App 上傳設定

  1. 車機和手機連到同一個 Wi-Fi。
  2. 找出電腦的區網 IP，例如 192.168.1.8。
  3. App 進入「上傳設定」。
  4. 上傳 API 填：http://192.168.1.8:8787/api/upload/{filename}
  5. 一次性連結 API 填：http://192.168.1.8:8787/api/one-time-links
  6. 傳輸格式選 PUT Binary。
  7. 依需求選「允許下載」或「只觀看」。

三、車機提交影片

  1. 進入「上傳分享」。
  2. 選擇 MP4 影片。
  3. 按「上傳並產生 QR」。
  4. App 會自動切到「一次 QR」。
  5. 手機掃描二維碼即可看影片；若允許下載，頁面會有下載按鈕。

四、正式上線注意

正式一次性 QR 必須放在真正的雲端後端，例如 NAS、公司伺服器、S3/R2 搭配 API、VPS 或雲端函式。 純前端頁面只能做測試 QR，不能保證只能看一次。

五、建置 APK

cd C:\Users\Administrator\ReplayCenter
npm run build
npx cap sync android
cd android
.\gradlew.bat assembleDebug
