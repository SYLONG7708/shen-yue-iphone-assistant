# 申悅車機助手 iPhone 手機版 專業商業版

這是一個 iPhone Safari 可加入主畫面的商業版 PWA App。上傳網站後，客戶可從 iPhone 主畫面用接近 App 的全螢幕體驗開啟。

## 主要功能

- 一鍵 LINE 聯絡申悅
- 一鍵撥打電話
- 一鍵導航到店
- YouTube 頻道入口
- Facebook 粉絲頁入口
- 車載安卓機教學
- 保固與安裝紀錄，本機儲存
- 主機規格新版選項
- 其他產品類別自行輸入
- 離線快取

## iPhone 使用方式

1. 把本資料夾整個上傳到網站主機。
2. 用 iPhone Safari 開啟網站網址。
3. 點選 Safari 分享按鈕。
4. 選擇「加入主畫面」。
5. 名稱使用「申悅車機助手」，按「新增」。

## 本機測試

在此資料夾用瀏覽器開啟 `index.html` 可看畫面。

若要測試離線快取，需透過網站或本機伺服器開啟，不能只用檔案路徑測試 service worker。
