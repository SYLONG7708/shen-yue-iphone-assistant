# 申悅車機助手 1.0.17 回放中心雲端版

## 建議安裝

- `申悅車機助手-1.0.17-雲端版-debug.apk`
- 英文檔名備用：`ShenYueAssistant-1.0.17-live-cloud-debug.apk`

此版本啟動後讀取：

`https://sylong7708.github.io/shen-yue-iphone-assistant/`

後續只要更新 GitHub Pages 內容，即可同步回放中心，不必重新安裝 APK。

## 本次新增

- 新增回放中心分頁與首頁快捷。
- 回放中心可上傳影片、產生一次性 QR、手機觀看與下載。
- 隱藏測試 API 設定與 WebView 相容模式標籤。
- 新增「分享通訊APP」，可呼叫 Android 系統分享視窗，選擇 LINE、微信、Messenger 等 App。

## 內建測試版

- `申悅車機助手-1.0.17-內建版-debug.apk`
- 英文檔名備用：`ShenYueAssistant-1.0.17-built-in-debug.apk`

內建版會把網頁資源包進 APK，適合無網路或本地驗證；正式雲端更新請使用雲端版。
